# Shambhavi Rai — Portfolio (React + Vite)

A premium, single-page developer portfolio built with React (Vite), Framer Motion,
and EmailJS. Dark "developer console" theme with light mode toggle.

## 📁 Features

-Responsive Design
-Dark & Light Theme 
-Smooth Animation
-Projects Showcase 
-Skills Section
-Experience Section
-Contact Form



## Tech Stack
-React
-Vite
-JavaScript
-CSS
-Framer Motion


## ⚙️ Installation

```bash
npm install
npm run dev
```


## 🏗️ Build for production

```bash
npm run build
```
